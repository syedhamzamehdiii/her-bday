A frontend-only romantic birthday experience built with **Next.js (App Router)** + **React** + **Framer Motion**.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000/birthday`.

## Deploy (Vercel)

Deploy directly as-is (no backend, no env vars required).
